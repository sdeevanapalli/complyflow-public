# Open Source Guide for ComplyFlow

## Security & Privacy
- **No credentials, secrets, or sensitive data are included in this repository.**
- Use `.env.example` and `credentials.json.example` to create your own configuration files.
- Never commit your real `.env` or `credentials.json` files to version control.

## Configuration
- Copy `.env.example` to `.env` and fill in your values.
- Copy `credentials.json.example` to `credentials.json` and add your Google Cloud service account credentials.
- For frontend, copy `frontend/.env.example` to `frontend/.env` and set your API and OAuth values.

## Reporting Security Issues
- Please report any security vulnerabilities privately to the maintainers.
- Do not disclose security issues publicly until they are resolved.

# Data Directory

This directory stores:
- **live_downloads/**: Real-time downloaded regulatory documents
- **raw_pdfs/**: Original PDF files for ingestion
  - **acts/**: Legal acts and regulations
  - **circulars/**: Tax circulars and notifications
  - **notifications/**: Official government notifications

## For Open Source Users

- This directory is used for document storage and processing.
- Add your own legal documents here for ingestion into the knowledge base.
- See the main README.md for instructions on document ingestion.

**Note**: No actual documents are included in this repository for security and licensing reasons.

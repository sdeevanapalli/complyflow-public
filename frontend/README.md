# ComplyFlow Frontend

This is the React + Vite frontend for ComplyFlow.

## Security & Open Source Notice

- No credentials or secrets are included in this repository.
- Use the provided `.env.example` to create your own `.env` file.
- Never commit your real `.env` file to version control.

## Getting Started

1. **Clone the repository**
2. **Install dependencies**
   ```sh
   npm install
   ```
3. **Copy the example environment file**
   ```sh
   cp .env.example .env
   # Edit .env and fill in your API URL and Google OAuth Client ID
   ```
4. **Run the development server**
   ```sh
   npm run dev
   ```

## Project Structure
- `src/` - Main source code
- `public/` - Static assets
